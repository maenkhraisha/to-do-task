## Getting Started

**First open VS code**<br/>
**From the VS code terminal navigate to the folder where you want to download the project**<br/>
```
$ git clone git@github.com:maenkhraisha/todo-list-application.git
```
```
npm install json-server
```
```
npm run server
```
``` 
npm install 
```
```
 npm start
```
